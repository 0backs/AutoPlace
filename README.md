#
씨앗을 소모하는 작물을 채집 시 자동으로 심어주는 플러그인입니다.
</div>

## 설치 방법
AutoPlace.jar 플러그인을 plugins 폴더 안에 배포해 주세요.

## 버전 안내
1.21.4 버전을 기준으로 제작되었으며, 특별한 능력이 없기에 상위 버전은 호환될 수도 있습니다.

## 권한
##### autoplace.use = 자동심기 명령어 사용 권한

## 명령어
/자동심기 - 기능을 활성화 또는 비활성화합니다.
